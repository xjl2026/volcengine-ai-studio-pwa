使用帮助

1. 获取 API Key

火山引擎控制台 → 火山方舟 → API Key 管理

2. 开通模型

在方舟控制台开通 Seedream(图片) 和 Seedance(视频)

3. 多 Key 管理

可保存多个 API Key，通过下拉框切换

4. 跨设备同步

在设置中填写 Supabase 信息和同步密钥，所有设备填同样的密钥即可同步历史记录。数据加密存储，无密钥无法解密。

5. 数据安全

API Key 仅保存在本地。同步的历史记录使用 AES-GCM 加密，即使数据库泄露也无法读取。

6. API Key 安全须知

API Key 是您的付费凭证，请勿泄露给他人。本应用仅将 Key 存储在浏览器本地，不会上传到任何第三方服务器。测试连接时会向火山方舟 API 发送一次最小请求以验证 Key 有效性。如怀疑 Key 泄露，请立即前往火山引擎控制台轮换 Key。
